# Humans API

Простое REST API для управления людьми (человеческими сущностями) в базе данных.
Для запуска docker-compose up --build

## Технологии

- **Go (net/http)**
- **PostgreSQL**
- **SQLC** - генерация запросов к бд.
- **Goose** — миграции бд.
- **Docker & Docker Compose**
- **.env**
- **Swagger (swaggo/http-swagger)**
